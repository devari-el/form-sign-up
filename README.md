## Instalação:

1. Faça um fork do repositório.

2. Clone sua fork:

```
git clone https://github.com/<USERNAME>/form-sign-up.git
```

2. Entrar no diretório:

```
cd form-sign-up
```

3. Instalar dependências:

```
npm install
```

4. Abrir o servidor de desenvolvimento:

```
npm run dev
```
